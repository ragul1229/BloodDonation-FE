# Blood Donation App (Frontend) 🩸

A responsive frontend for a blood donation system allowing users to request or donate blood.

## 🚀 Features
- Blood request form
- Donation info entry
- Responsive and accessible layout

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
